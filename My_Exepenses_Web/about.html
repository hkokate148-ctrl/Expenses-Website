About Expense Tracker

Expense Tracker is a simple and user-friendly web application designed to help users manage their daily expenses easily. Users can register, login, add expenses, view records, and delete unnecessary entries. This project is created to make expense management simple, fast, and organized for everyone.

The website is built using HTML, CSS, JavaScript, PHP, and MySQL. It provides a responsive design that works smoothly on desktop and mobile devices.
