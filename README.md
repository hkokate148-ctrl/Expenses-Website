# Expenses-Website
Developed an Expense Tracker web application to help users manage and monitor their daily expenses efficiently. The system allows users to add, update, delete, and track expense records with a simple and user-friendly interface. Built using PHP, MySQL, HTML, CSS, JavaScript, and Bootstrap for responsive performance and smooth user interaction.

# Technologies Used:
PHP
MySQL
HTML5
CSS3
JavaScript
Bootstrap

# Key Features:
Add & Manage Expenses
Expense Record Tracking
Update & Delete Functionality
User-Friendly Dashboard
Responsive Interface
Database Integration
